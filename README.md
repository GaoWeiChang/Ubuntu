## sudo su
  * To switch from sudo to root.
  * "root" is the actual name of administration account.
  ```
  thaksin5@LAPTOP-9JB425LQ:/mnt/c/Users/CHANG$ sudo su
  root@LAPTOP-9JB425LQ:/mnt/c/Users/CHANG#
  ```
